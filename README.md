# For Tyler

To use this project:
1) git clone the repository to somewhere on your system.
2) Once inside the project, create a virtual environment with `python3 -m venv .venv`
3) Then, activate the virtual environment with with `.\.venv\Scripts\activate.bat`
4) Install necessary dependencies with `pip3 install -r requirements.txt`
5) Then, run `python3 main.py` to run the file.
  a) Be warned, it takes a bit to run the first time as these are big libraries, so it's loading a lot into RAM. It's a lot faster after the first time you run main.py
6) Stay lookin pretty

If you need any help or there's issues, reach out, I'm open any time.
